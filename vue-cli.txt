1.node 安装 8版本以上   node -v  查看版本号 是否安装完毕及版本
3.npm install -g @vue/cli     安装vue-cli 脚手架
4.vue --version    查看vue版本 是否3以上
5.vue create project（项目名字）   创建项目



TypeScript 支持使用 TypeScript 书写源码
Progressive Web App (PWA) Support PWA 支持。
Router 支持 vue-router 。
CSS Pre-processors 支持 CSS 预处理器。
Linter / Formatter 支持代码风格检查和格式化。
Unit Testing 支持单元测试。
E2E Testing 支持 E2E 测试。
6.运行   npm run serve
7.打包  npm run build
8.引入插件npm install axios --save
9.npm install 安装项目需要的插件
